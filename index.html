About the Launcher

Top Flash Games Arcade v1.1 is developed by CyberVerse Computer Training. This application allows users to play their favorite childhood Flash games directly on Windows PC offline, without needing any web browser or insecure active plugins. Perfect for archiving and nostalgic gaming.

What's New & Key Features

Dark/Light Theme Toggle

Switch between Dark Mode and Light Mode in real-time. (အမှောင်/အလင်းမုဒ် ခလုတ်တစ်ချက်နှိပ်ရုံဖြင့် ပြောင်းနိုင်)

Myanmar Language

Full Myanmar (Burmese) language interface support. (မြန်မာဘာသာစကား အပြည့်အဝ ပံ့ပိုးမှု)

Parental Control System

Secure PIN-protected mode to hide games for child safety. (PIN ကုဒ်ဖြင့် လုံခြုံစွာ ကလေးများအတွက် ဂိမ်းများဖျောက်နိုင်)

Players' Choice (Favorites)

Bookmark and organize your favorite games. (ကြိုက်နှစ်သက်ရာ ဂိမ်းများကို သီးသန့်စုစည်းနိုင်)

Drag & Drop Import

Simply drag SWF files into the app to add games. (SWF ဖိုင်များကို အပေါ်သို့ဆွဲချရုံဖြင့် အလိုအလျောက်ထည့်သွင်း)

Auto-Thumbnail Capture

Automatically captures game screenshots when playing. (ဂိမ်းကစားသည့်အခါ Screenshot များကို အလိုအလျောက်ဖမ်းယူ)

How to Use

  1. Download and run TopFlashGames_Setup_v1.1.exe
  2. Follow the installation wizard instructions.
  3. Launch the application from Desktop or Start Menu.
  4. Drag & drop your .swf files or use "Add Game" button.
  5. Enjoy your childhood memories!
